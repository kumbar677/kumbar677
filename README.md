## ✦ What I Do

🔹 **Prompt Engineering** — Crafting precise, structured prompts for LLMs (GPT, Claude, Gemini) to unlock maximum accuracy and relevance in outputs.

🔹 **LLM Optimisation** — Applying chain-of-thought prompting, few-shot learning, context management, and output validation to make AI models more reliable and consistent.

🔹 **Full Stack Web Development** — Building end-to-end web applications with clean frontends (HTML, CSS, JS, React) and solid backends (Node.js, Python, REST APIs).

🔹 **AI-Powered Interfaces** — Integrating LLM features into real products — chatbots, smart search, content generation tools, and intelligent assistants.

🔹 **Problem-First Mindset** — Every prompt and every line of code starts with one question: *what problem does this solve?*

🔹 **Clean & Scalable Code** — Writing readable, maintainable code with a focus on structure, performance, and long-term scalability.
