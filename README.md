## ✦ What I Do

🔹 **Prompt Engineering** — Crafting precise, structured prompts for LLMs (GPT, Claude, Gemini) to unlock maximum accuracy and relevance in outputs.

🔹 **LLM Optimisation** — Applying chain-of-thought prompting, few-shot learning, context management, and output validation to make AI more reliable.

🔹 **Full Stack Web Development** — Building end-to-end web apps with clean frontends (HTML, CSS, JS, React) and solid backends (Node.js, Python, APIs).

🔹 **AI-Powered Interfaces** — Integrating LLM features into real products — chatbots, smart search, content generation tools.

🔹 **Problem-First Mindset** — Every prompt and every line of code starts with one question: *what problem does this solve?*
